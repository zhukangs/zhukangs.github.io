导语

在做完根据Excel模板导出数据后，就对Word模板导出产生了兴趣。其实原理差不多。所以还是选择的PHPOffice 套餐。

PHPWord是一个用纯PHP编写的库，它提供了一组用于写入和读取不同文档文件格式的类。当前版本的PHPWord支持Microsoft Office Open XML（OOXML或OpenXML），Office应用程序的OASIS Open Document Format（OpenDocument或ODF）和RTF格式（RTF）。

PHPWord文档传送门

{.figure}

开始

安装

composer require phpoffice/phpword

使用

拿到原始的模板还需要做一些简单的修改：

{.figure}

代码

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;
use PhpOffice\PhpWord\PhpWord;
use PhpOffice\PhpWord\TemplateProcessor;


class TestController extends Controller
{
    .
    .
    //Word

    public function wordExport(Request $request)
    {
        $template = new TemplateProcessor(public_path().'/phpoffice/word_temp.docx');
        //设置替换值 （只能替换纯文本）
        $template->setValue(
            ['Name','Date','Address','Organizer','Activity'],
            [$request->name,$request->start_time,$request->address,$request->organizer,$request->activity]
        );
        $filename = date('Y-m-d', time()).rand(1000, 9999). ".docx";
        //保存到项目
        $template->saveAs(public_path().'/phpoffice/'.$filename);

        //建议：将数据['Name','Date','Address','Organizer','Activity']写一份到数据库
        //TODO

        return back()->with(['success'=>'导出成功','filename'=>$filename]);
    }

}
